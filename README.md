# Weekly Planner

## Goals

- CRA 없이 웹팩, 리액트, 타입스크립트 개발 환경 구축
- Web Storage API
- Custom hooks
- Memoization
- 반응형 디자인

## Live Demo

[Link](https://sewook-weekly-planner.netlify.app)

## Tech and Libraries

- Webpack
- React
- TypeScript
- Emotion(Styled Components)
- Web Storage

## Getting Started

1. Clone this repo

```
git clone https://github.com/SewookHan/countries.git
```

2. Install

```
yarn
```

3. start

```
yarn start
```
